
# Medi Score

### DISCLAIMER:
### **This product is _not_ to be used to provide for medical advice; it is presented purely for simulation purposes!**

Medi Score is a program written in Java which depicts a system by which medics could assess a patient's physiological measurements.

The program is intended to fulfil the criteria laid down by Aire Logic in their technical test for junior candidates.

It will run you through the process of recording readings such as:

    - Whether the patient is on air or oxygen_
    - Their level of consciousness
    - Rate of respiration
    - Oxygen saturation
    - Temperature
    - Blood glucose level.
You will be given a simple score for the patient, which represents how much these markers deviate from a healthy baseline.
